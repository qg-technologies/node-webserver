# WEB SERVER

Creación de servidor rest para peticiones GET, PUT, PATCH, DELETE,

El servidor express se ejecutará en el puerto 3000 cuando sea ejecutado localmente.

### Pre-requisitos

- Intalar express
- body-parser
- Instalar MongoDB
- Instalar Robot3T

## Construido con 

* Node.js
* MongoDB
* Robo3T
